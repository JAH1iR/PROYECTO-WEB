# 🌿 Turismo Cañazas de Capira - PROYECTO-V1

Este es un sitio web moderno diseñado para promocionar el turismo y las bellezas naturales de **Cañazas de Capira**. El proyecto utiliza una estructura limpia y escalable para facilitar el crecimiento futuro.

## 🚀 Características
* **Diseño Modular:** CSS separado por funciones (Base, Componentes y Páginas).
* **Organización Profesional:** Estructura de carpetas optimizada para multimedia y archivos HTML.
* **Responsive:** Preparado para adaptarse a diferentes dispositivos.

## 📁 Estructura del Proyecto
* `/ALL-HTML5`: Contiene las subpáginas (Fincas, Quebradas, etc.).
* `/CSS3`: Archivos de estilos (`style.css`, `COMPONENTES.css`, `ALL-HTML.css`).
* `/MEDIA`: Almacén de imágenes y videos organizados por categorías.
* `index.html`: La puerta de entrada principal del sitio.

## 🛠️ Tecnologías
* **HTML5**
* **CSS3** (Flexbox y CSS Grid)
* **Git & GitHub** para control de versiones
